* 학회 과정을 수료하기 위해서 상점을 늘리고, 벌점을 줄이는 것이 필수적이다. 특히, 우수 학회원이 되기 위해서는 상벌점 관리가 더욱 중요하다.
* 그러나, 대부분의 학회원들이 상점보다 벌점이 많은 상태이다. 학회 내 상점 데이터셋을 통해 (상점 - 벌점)의 값이 0 이상인 학회원들을 예측하고, 적절한 인사이트를 도출하기로 결정했다.
